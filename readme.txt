Discount Calculator

App with two views, a calculator and results view.  Calculator view allows user to enter a price, dollars off the price, a discount percent, another discount percent, and tax percent.  A calculate button, once pressed will calculate original price and discount price.

When user swipes left on screen, it will bring them to results view.  It shows a graphical view of the costs.  User can swipe between two views.
